## Day - Exam Countdown App
Tracks exam dates and reminds the user up to date
## Features
- Add subjects and exam schedule
- Sends notification based on the exam schedule
- Has a pomodoro counter in built
- User can create their own exam preparation schedule
## Built With
HTML, CSS, JavaScript - built using Bolt.new + Claude.AI
